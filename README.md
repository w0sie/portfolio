# Narrative Game Development Portfolio

A personal portfolio website showcasing narrative game development work, including interactive storytelling, worldbuilding, and game design projects.

## Live Site

This portfolio is hosted on GitHub Pages at: [https://w0sie.github.io/portfolio/](https://w0sie.github.io/portfolio/)

## Features

- **Home Page**: Introduction and purpose of the portfolio
- **Portfolio Page**: Showcase of narrative game projects, writing samples, and prototypes
- **About Me Page**: Background, skills, and approach to narrative design
- **Contact Page**: Multiple ways to get in touch

## Design Philosophy

The site features a clean, modern design with a narrative-focused aesthetic:
- Elegant typography using serif fonts for headings
- Professional color palette with purple accents
- Responsive layout that works on all devices
- Accessible design following WCAG guidelines

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Static site structure optimized for GitHub Pages

## Local Development

To view the site locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. Or use a local server: `python -m http.server 8000`

## Customization

To customize this portfolio for your own use:

1. **Update Content**: Replace placeholder text in all HTML files with your own information
2. **Add Images**: Replace placeholder images in the `/images` folder with your own screenshots and photos
3. **Modify Styling**: Edit `css/style.css` to adjust colors, fonts, and layout
4. **Contact Form**: Connect the contact form in `contact.html` to a backend service (e.g., Formspree, Netlify Forms)

## Adding New Projects

To add new projects to the portfolio:

1. Open `portfolio.html`
2. Copy an existing `<article class="portfolio-item">` block
3. Update the content, tags, and image source
4. Save and refresh
